Examples and patterns

Paragraph 1.

Web of Things Architecture 1.1 is used in smart homes with where devices communicate through the internet. As an example, a smart speaker like Alexa can connect to the smart lights in your home and disconnect them if need be.

Paragraph 2.

Developers use WoT Architecture 1.1 in tandem with technologies like HTP and JSON so that devices can share information. As an example, smart watches can send heart rate data to a phone application linked to the device through a web based communication.

Paragraph 3.

The architecture alsu uses the Thing Description to explain how exactly a device may work. An example would be a smart Thermostat, it can describe its temperature controls to a mobile application so it can automatically find the device and change the remperature remotely (from work or the grocery store)
