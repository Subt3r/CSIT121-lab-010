What is the Web of Things (WoT) Architecure 1.1?

Paragraph 1.

Web of Things (WoT) Architecture 1.1 is an official standard that was designed specifically to fix the issue of fragmentation of the internet oif Things (IoT). It enables interoperabilit across IOT platforms and application domains. The goal of WoT would be to preserve and go along with the already existing IoT standars and solutions out there. WoT Architecture is designed to describe what already exists, and can only create new mechanisms when absolutely necesssry.

Paragraph 2.

The 1.1 architecture builds on top of the original 1.0 model by adding advanced discovery mechanisms and protocol supports. Its framework has three main parts being thing description (TD) which is the core of the WoT. it acts as machin readable JSON to describe a devices metadata propertiess, its actions, and even secuirty requirements. It also uses protocol binding templates that define how to acces and interact with devices using W/IoT protocols.

Paragraph 3.

It is mainly used for smart homes, wearable devices like a smart watch, healthcare systems, and IoT devices.
